## Awesome RTC Electron App

#### Build
- Mac
```bash
npm install
./node_modules/.bin/electron-rebuild
```

- Windows (in vusal studio command prompt)
```bash
npm install
#or for visualstudio 2013
npm install --msvs_version=2013
.\node_modules\.bin\electron-rebuild.cmd
```
#### Run
```bash
npm start
```
